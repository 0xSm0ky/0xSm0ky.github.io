أهلاً وسهلاً. هذه زاوية مظلمة وبسيطة من الإنترنت أكتب فيها ملاحظات عن البرمجة، والأمن السيبراني، والهندسة العكسية، وما يخطر على البال من تجارب جانبية.

## كيف تعمل المدونة؟

- التدوينات ملفات `.md` عادية تحت `src/content/blog/`.
- أكتبها داخل **Obsidian** مع فتح المجلد كـ Vault.
- كل `git push` إلى الفرع `main` يُشغّل GitHub Actions ويُحدّث الموقع تلقائيًا على GitHub Pages.
